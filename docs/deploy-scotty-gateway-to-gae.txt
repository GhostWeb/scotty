Howto deploy scotty-gateway to Google Appengine:

1. Update Version Number in scotty-gateway\src\main\webapp\WEB-INF\appengine-web.xml
2. Run appcfg update to deploy to gae:
<Google-Appengine-SDK>/bin/appcfg update <Path-to-target-scotty-gateway-war>

sample:
D:\springsource\sts-2.9.2.RELEASE\plugins\com.google.appengine.eclipse.sdkbundle_1.7.2.1\appengine-java-sdk-1.7.2.1\bin>appcfg update C:\Users\flo\git\scotty\scotty-gateway\target\scotty-gateway
